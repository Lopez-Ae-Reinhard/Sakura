后端
    服务端:
        创建多进程
        处理请求
        数据处理-->交给链接数据库模块
        反馈结果

    数据库模块:
        数据操作
        提供结果


前端
    客户端
        请求发起
        结果展示
        界面跳转


功能
    历史记录
        客户端:
            * 发送请求
            * 循环接受历史记录

        服务端:
            * 接受请求
            * 查询历史记录
            * 发送历史记录(前10条)

    查单词
        客户端
            * 循环输入单词,
            * 发送给服务器
            * 获取结果
            * ## 退出

        服务端
            * 接受请求
            * 查找单词
            * 将结果发送给客户端
            * 插入历史记录

    协议
        注册 R name password
        登录 L name password
        退出 E
        查词 Q name word

    注册
        客户端
            *输入注册信息
            *将信息发送给服务端
            *等待反馈

        服务端
            *接收注册信息
            *验证用户是否存在
            *则插入数据库
            *反馈信息(成功or失败)
